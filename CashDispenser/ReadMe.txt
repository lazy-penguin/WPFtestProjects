Cash Dispencer — Банкомат

Описание работы:
Программа имитирует взаимодействие с банкоматом в рамках одного пользователя.
Банкомат может работать с купюрами номиналом в 10, 50, 100, 200, 500, 1000, 2000, 5000.
Изначально в банкомате лежит 40 купюр - по 5 купюр каждого номинала.
Максимальная общая вместимость купюр - 100 экземпляров.

Пользователю доступны следующие опции:
* Balance — отобразить текущий баланс (изначально - 0).
* Put — внести деньги.
* Withdraw — снять деньги, можно выбрать желаемый номинал купюры.
* State — отобразить внутреннее состояние банкомата.
* Back —  вернуться к главному экрану.
* Exit — закончить работу.

Необходимые программные средства:
* Операционная система Windows.
* Версия .NET Framework 4.7.2 или выше.
