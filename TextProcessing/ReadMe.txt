Test Processing — Обработчик текста

Описание работы:
Для начала процесса обработки пользователю необходимо:
* Выбрать исходный и выходной файлы.
* Ввести числовой порог. Слова, у которых длина меньше порогового значения, удаляются.
* Задать, нужно ли удалять знаки преминания.
Кнопку "Process" запускает процесс обработки.
Кнопка "New Input" очищает поля и позволяет ввести новые данные.
На правой панели отображаются текущие процессы обработки.

Необходимые программные средства:
* Операционная система Windows.
* Версия .NET Framework 4.7.2 или выше.
